# Product Spec

## One-line Description

Packed helps users manage reusable packing groups and check items for the day.

## Core Value

- Reduce forgotten items through repeatable daily checklists.
- Keep packing setup lightweight.

## Target Users

- People preparing for school, work, travel, gym, or daily routines.

## Core Operation

- Create a group, add items, check today's packed state, and optionally get reminders.

## Free Features

- Packing groups and items.
- Daily check state.
- Settings and local reminders.

## Paid Feature Candidates

- Unknown. Possible candidates: unlimited groups, templates, themes.

## Non-goals

- Inventory management.
- Shared household/team packing.
- Server sync in v1.
